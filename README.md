# replit-exports

A collection of exported Replit projects.

## What is this?

This repo is a backup of multiple projects exported from Replit, kept together for reference.

## Quick start

Clone the repo and fetch large media files (Git LFS):

```bash
git clone https://github.com/mellownightpirate/replit-exports.git
cd replit-exports
git lfs install
git lfs pull
```
